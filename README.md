# Restaurant Website

A responsive restaurant website inspired by The Cheesecake Factory, built using HTML, CSS, Bootstrap, and JavaScript. The website showcases menu items, restaurant information, customer reviews, team members, reservation forms, and social media integration.

## Features

- Responsive navigation bar
- Full-screen hero image carousel
- About section
- Interactive menu categories
- Services section
- Team member profiles
- Customer reviews carousel
- Reservation form
- Instagram gallery section
- Social media links
- Responsive design for desktop and mobile devices

## Technologies

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Remix Icons
- Google Fonts

Open `index.html` in your browser.
